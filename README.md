# Automation Website Testing

##  Project Description
This project is an automation testing framework for AutomationExerciseWebsite
It is developed using Selenium WebDriver and follows a structured test automation approach.

##  Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- IntelliJ IDEA

##  Project Structure
- src/main/java → Page classes / utilities
- src/test/java → Test cases
- pom.xml → Project dependencies

##  Test Scenarios Covered
- Login
- Validations
- Add to cart functionality
- Checkout process

## How to Run the Project
1. Clone the repository
2. Open in IntelliJ IDEA
3. Install dependencies using Maven
4. Run test classes from `src/test/java`

## Key Features
- Page Object Model 
- Reusable methods
- Clean test structure
